Panorámicas

Posted on sáb 05 julio 2008


Muchas veces al intentar fotografiar algún paisaje o alguna situación, simplemente nuestra cámara no nos permite congelar la totalidad, motivo por el cual se pierde el contexto necesario para darle valor a esa toma.
Las panorámicas vienen a solucionar este problema, permitiendo fotografiar un ángulo mayor del campo visual (gran angular). Esto lo podemos lograr mediante una cámara que soporte la toma de panorámicas de gran ángulo o mediante una serie de fotografías normales que capturen el campo visual deseado y unirlos mediante algún software.
Para los que emplean software libre les dejo una lista de programas que pueden usar:

  • Hugin
  • Panotools
  • Enblend
  • Autopano-sift
  • PanoPoints
  • The Gimp (la manera difícil para los masoquistas)
