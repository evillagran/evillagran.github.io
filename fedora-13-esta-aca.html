Fedora 13 está acá!!

Posted on jue 27 mayo 2010

Luego de esperar una semana extra para el lanzamiento de Fedora 13 ya podemos disfrutar de este nuevo release. Como siempre, incluye una serie de nuevas características entre las que sobresalen:

  • Instalación de controladores de impresión automática, lo que facilitará la vida a los usuarios que requieren imprimir.
  • Instalación de paquetes de idioma
  • Administración de perfiles de colores para dispositivos, permitiendo una representación fiel de los coloren entre dispositivos como monitores, impresoras, etc.
  • Actualizaciones para NFS
  • Más avances en Btrfs, como recuperación de sistemas de archivos vía snapshots.
  • Python 3 convive con Python 2.x, permitiendo correr una serie de programas que dependen de Python 2.x y además hacer cosas con Python 3.
  • Netbeans cumple las specs. de Java EE 6.
  • Línea de comandos para NetworkManager
  • Indicador de intensidad de señal y roaming para conexiones con tecnologia celular (GPRS/EDGE/3G).
  • Cliente Pino para twitter.
  • Mejoras para virtualización con KVM

Más detalles pueden ser vistos en las notas de lanzamiento. Si desean descargarlo pueden ver las diversas opciones que hay en http://fedoraproject.org/es/get-fedora

En mi caso particular, instalé Fedora 13 en tan solo 7 minutos utilizando un pendrive como instalador. Pueden encontrar más info en http://www.proyectofedora.cl/2010/05/fedora-13-instalado-en-7-minutos/
