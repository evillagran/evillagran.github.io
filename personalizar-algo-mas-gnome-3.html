Personalizar algo más GNOME 3

Posted on dom 24 abril 2011

Siempre es deseable personalizar el escritorio, en el caso de GNOME 3 hay muchas cosas que han quedado en el tintero.
Si deseamos realizar configuraciones sobre tema visual, tamaño de letras, etc. debemos instalar gnome-tweak-tool.

Además podemos emplear las extensiones (éstas son las disponibles en F15 beta):

  • gnome-shell-extensions-alternate-tab : Classic Alt+Tab behavior. Window based instead of app based
  • gnome-shell-extensions-alternative-status-menu : For those who want a power off item visible at all the time
  • gnome-shell-extensions-auto-move-windows : Assign specific workspaces to applications
  • gnome-shell-extensions-common : Files common to GNOME Shell Extensions
  • gnome-shell-extensions-dock : Shows a dock-style task switcher permanently
  • gnome-shell-extensions-user-theme : Lets the user select a custom theme for the shell
  • gnome-shell-extensions-windowsNavigator : Keyboard selection of windows and work-spaces in overlay mode
